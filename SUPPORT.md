# Support

## How to get help

- **For bugs or feature requests** in an MVNG Consulting repository, open a GitHub issue using the appropriate template.
- **For general enquiries** about MVNG, email **hello@mvng.nz** or visit [mvng.nz](https://mvng.nz).
- **For security issues**, please see [`SECURITY.md`](SECURITY.md) and do not open a public issue.

## Commercial support

MVNG Consulting provides commercial architecture, cloud, AI, and integration consulting. For engagement enquiries, reach out via [mvng.nz](https://mvng.nz) or **hello@mvng.nz**.
